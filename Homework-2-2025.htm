Network Programming Homework (2)

作業繳交通則

A. 以本校教學平台繳交

B. 逾期繳交不予以計分

C. 主程式開始時需以註解說明下列相關資訊:

(1) 作者姓名及學號. 若參考他人程式改寫, 請加註原創者姓名及學號

(2) 程式執行時的操作說明

(3) 符合的評分標準及自評應得的分數, 注意: 評分時會參考此資訊, 請仔細正確填寫

(4) 若參考他人程式, 請說明不同的地方或增加的功能

(5) 其他有利於評分的說明, 例如獨特的功能等

D. 明顯相似別人程式(包含從網路下載的程式), 無論有無註明原創者資料及不同之處, 均視為抄襲

E. 辨認原創者並不是老師的責任, 所以程式明顯相似者均視為抄襲

F. 若使用AI輔助撰寫程式, 必須於註解中說明提供的prompt以及修改的過程, 否則若有其他明顯相似程式均視為抄襲

作業題目

1. 以 PHP 寫一個 MicroSoft Windows 11 上的標準小算盤.

輸入: 依照小算盤

輸出: 依照小算盤

使用語言: PHP + HTML

繳交日期: 參考本校教學平台

分數比重: 學期總分 3 分

繳交內容:

1. 本作業需僅用一個 PHP 程式完成, 不可以 Java Applet 或 JavaScript 或直接呼叫外掛程式完成; 需命名為 cal-XXXXXXXX.php (XXXXXXXX代表學號).

2. 程式修改部分需加入註解.

評分標準(加分部分為可能最大值, 減分部分則為固定值):

1. 支援整數所有計算 (+80%)

2. 支援小數 (+20%)

3. 程式檔案沒有遵照規定命名 (–20%)

4. 主程式開始沒有包含作者、操作說明及符合的評分標準等資訊(即不遵守作業繳交通則C) (-20%)

5. 程式沒有適當註解 (–20%)

6. 抄襲 (上述評分不計, 直接 -50%)
